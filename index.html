How do I create a new event?

To set up a new event, navigate to the process page and select ‚ÄúCustomize Tabulation Button.‚Äù Enter the required details such as the Event Name, number of contestants, number of judges, and upload a contestant photo. After that, add the judges‚Äô names and contestants to get started.

How do judges submit their scores?

Judges sign in to their accounts, wait for the admin to deploy the event, and then access the list of contestants. They provide scores for each criterion, and the system instantly computes the overall totals in real time.

Can scores be edited after submission?

Once judges submit their scores for a contestant, they cannot make any changes. Therefore, ensure that all scores are finalized and accurate before submitting.

How do I add multiple judges?

Only admins have the ability to add judges. To do this, go to Customize Tabulation for customization, then enter the Event Name, Number of Contestants, and the Number of Judges. Judges themselves cannot add or change the number of judges.

Is the scoring system transparent?

Yes! The system displays scores in real time in the scoreboard page for all viewers. In addition, the admin dashboard generates clear, transparent tables showing each judge‚Äôs scores.
